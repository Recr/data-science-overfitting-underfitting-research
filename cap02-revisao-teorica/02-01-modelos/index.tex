\subsection{Modelos de Machine Learning}

O \textbf{\textit{Machine Learning}} é um campo da Inteligência Artificial (IA) que explora a construção de modelos computacionais capazes de identificar padrões, realizar previsões e tomar decisões com intervenção humana mínima e baseando-se em conjuntos de dados finitos.

Para a construção de um bom modelo de ML, deve-se coletar um conjunto de dados relevante para o problema a ser resolvido. Esses dados servirão como “combustível” para que um algoritmo selecionado possa aprender padrões em cima desse conjunto. Após estar devidamente treinado, o modelo poderá identificar padrões em novos dados e fazer previsões ou classificações com base no aprendizado.
